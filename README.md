# Random Password Generator

**Автор:** Катинова Динара  
**Вариант:** Random Password Generator

## Описание

Простая программа для генерации случайных паролей. Длина выбирается ползунком, можно включить буквы, цифры или спецсимволы. История сохраняется в файл passwords.json.

## Что нужно для запуска

- Python 3
- tkinter, json, random, string


## Как запустить

```bash
python main.py
